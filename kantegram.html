KanteGram

This was one of my favorite projects to work on because it allowed me to explore how front end and back end web development function together. Using Instagram's API, I was able to pull a users information and their media and post it into specific locations on the website. I kept the website simple and clean, and plan on adding additional features to the web app. The purpose of this project was to design how I think the desktop version of Instagram should look like. The current version lacks many important features and feels like it is more of a hassle to use compared to the web app.

However, there are certain limitations to using Instagram's API in a sandbox before being approved as a valid app. Because of this, my Instagram community is only limited to currently 3 people, 2 of which are my friends and the other being me. Besides that, the app still functions correctly and is able to make different requests from Instagrams server.
