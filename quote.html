Kanye Quotes

I completed this project as a part of FreeCodeCamp's curriculum. This project was also my first exposure to using an API. I was tasked with making sure that it generates random quotes and gives users the options of being able to tweet the quote to twitter.

I used Googles Material Design for the design of the project, and I chose to use Kanye West quotes because often times they can be very inspirational and motivating. The quotes come from a public API using a GET request. The quotes also get updated to the page without having to reload the web page.

This project no longer provides the user with quotes whenever they click the button because the API I was using has gone down.
